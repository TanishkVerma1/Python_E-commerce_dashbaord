# SouqPlus – UAE E-commerce Growth & Operations Dashboard

This repo is Streamlit Cloud–ready and implements **all required + optional** assignment requirements:

## Pages (toggle via sidebar)
- Executive View
- Manager View (includes drill-down + sortable problem table)
- What-If Analysis (required sliders)
- Customer Tiers (optional feature + optional filter)
- Data Quality & Cleaning (shows injected issues + cleaning steps)
- Dataset Explorer

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
1. Push to GitHub (include the `data/` folder)
2. Streamlit Cloud → New app → choose repo → main file: `app.py`
3. Done ✅
